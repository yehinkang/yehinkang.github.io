Toyota Highlander Pulley Tool

August 29, 2024
Tags: Hardware

This is a crankshaft pulley remover tool that I designed to be cut using a waterjet cutter and 2 flat peices of stock. The tool is designed to be used on a Toyota Highlander 3.5L engine to remove the crankshaft pulley. The tool is designed to be used with a 1/2" drive ratchet slot, and the thickness of the metal is 1/4" for the handle and 3/8" for the part in the pulley respectively. Both pieces are held together to form the tool using a pair of M8 bolts. The tool was used to hold a crankshaft pulley in place for a 2002 toyota highlander 3.0L V6, which enabled me to break the crank pulley bolt loose. This is a necessary step in replacing a timing belt for the Toyota MZ engines. In theory this design should work for all Toyota MZ and VZ engines, but crankshaft pulley bolt holes pacing should be double checked with the design to ensure that this will fit.
